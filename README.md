# ecommerce-django

ecommerce-django es un proyecto RESTful creado para demostrar mis conocimientos con Django y Python. Implemento vistas, servicios, serializers, y el ORM de Django para estructurar el backend. He integrado autenticación basada en JWT de manera personalizada, retirando por completo el sistema de auth y admin nativos de Django para tener un control total sobre el flujo de autenticación y permisos. Aunque no uso esos sistemas en este proyecto, tengo experiencia trabajando con ellos. Planeo seguir mejorando el proyecto continuamente.# ecomerce-django
